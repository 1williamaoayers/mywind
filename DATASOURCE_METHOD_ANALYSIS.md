## 失败原因分析

### HTTP 状态码分布

| 状态码 | 数量 | 含义 |
|--------|------|------|
| 200 但0条 | 11 | API正常但无数据/解析失败 |
| 404 | 3 | 接口地址变更 |
| 403 | 1 | 被禁止访问 |
| 429 | 1 | 请求频率限制 |
| 500 | 3 | 服务器错误 |
| 405 | 1 | 方法不允许 |
| 401 | 1 | 未授权 |

### 主要问题

1. **API 地址变更**: 格隆汇、港交所、经济通等返回404
2. **解析逻辑失效**: 11个源HTTP成功但解析出0条数据
3. **反爬限制**: 微博403、Yahoo 429
4. **周末无数据**: 港股通资金为交易日数据

3. SEC EDGAR - 官方 API 稳定

| 优先级 | 数据源 | 问题 | 修复方案 |
