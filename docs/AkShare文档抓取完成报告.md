## 🔧 技术实现

### 抓取脚本特性
- **工具**: Puppeteer + Turndown  
- **转换**: HTML → Markdown  
- **容错**: 自动处理失败页面  
- **延迟**: 500ms防止请求过快  
- **格式**: 保留表格/代码块/链接

### 遇到的问题及解决
1. **问题**: 缺少 `turndown` 模块  
   **解决**: `npm install turndown --save`
   
2. **问题**: URL拼接错误,缺少斜杠  
   **解决**: 修复URL拼接逻辑,第二次运行全部成功
