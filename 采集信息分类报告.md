# MyWind采集信息分类报告

**日期**: 2026-01-02

---

## 一、信息分类

项目采集的信息分为**三大类**：

| 类别 | 说明 | 数据源 |
|-----|------|--------|
| **新闻** | 实时资讯、个股动态、行业新闻 | 同花顺、富途、格隆汇、智通财经等 |
| **财报/公告** | 业绩公告、年报、季报、公司公告 | 同花顺、港交所HKEX |
| **研报** | 券商研究报告、分析师观点 | 东财研报、研报客、发现报告 |

---

## 二、各类别详情

### 2.1 新闻类

**定义**: 实时发生的市场消息、公司动态

**信息源**:
| 来源 | 类型 | 采集方式 |
|-----|------|---------|
| 同花顺7x24 | 实时快讯 | HTTP API |
| 富途 | 个股新闻 | Puppeteer |
| 格隆汇 | 港股资讯 | Puppeteer |
| 智通财经 | 港美股新闻 | Puppeteer |
| 雪球/微博/知乎 | 社交舆情 | Puppeteer |

**采集频率**: 高（每小时或更频繁）

---

### 2.2 财报/公告类

**定义**: 上市公司官方发布的财务报告和法定公告

**信息源**:
| 来源 | 内容 | 格式 |
|-----|------|------|
| 同花顺 | 业绩公告、季报、年报 | PDF |
| 港交所HKEX | 法定公告、业绩公告 | PDF |
| 新浪财经 | 财务数据摘要 | HTML/DOM |

**内容包括**:
- 季度业绩公告（Q1/Q2/Q3/Q4）
- 中期报告（半年报）
- 年度报告
- 股息公告
- 重大事项公告

**采集频率**: 中（每日1-2次）

---

### 2.3 研报类

**定义**: 券商、机构发布的研究分析报告

**信息源**:
| 来源 | 内容 | 格式 |
|-----|------|------|
| 东财研报 | 券商研报 | PDF链接 |
| 研报客 | 综合研报搜索 | PDF链接 |
| 发现报告 | 行业研报 | PDF链接 |

**采集频率**: 低（每日1次）

---
